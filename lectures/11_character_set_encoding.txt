
    Character Set and Encoding

        Binary Data
            คอมพิวเตอร์จัดเก็บข้อมูลในรูปแบบของ binary โดยจะเป็นกลุ่มของตัวเลข 0, 1

        คอมพิวเตอร์จะแปลง ตัวอักษร เป็น ตัวเลข (character code)
        เช่น v -> 86

        Character Set 
            เป็นกลุ่มของตัวอักษรที่ถูกกำหนดไว้ล่วงหน้ากับตัวเลข
            โดยที่นิยมใช้ คือ Unicode และ ASCII

        UTF
            เป็นการระบุว่าอักขระควรจะเป็นเข้ารหัสแบบ byte
            เช่น UTF-8 คือใช้ 8 bit
