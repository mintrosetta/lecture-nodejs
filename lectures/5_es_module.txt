
    CommonJS
        แต่ละไฟล์ถือเป็นไฟล์ module
        variable, functions, classes ไม่สามารถเข้าถึงจากข้างนอกได้โดยตรง
        ต้องระบุให้ชัดเจน ว่าอะไรบ้างที่จะ export ออกไปด้วย module.export 
        ใช้ require() ดึงไฟล์ที่ต้องการใช้มา

    ES Module 
        เป็นมารตรฐานใหม่ ทำให้ code เป็นระเบียบมากขึ้น
