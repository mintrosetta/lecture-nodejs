Modules
    จะห่อหุ้ม code ของเรา และทำให้สามารถนำกลับมาใช้ใหม่ได้
    โดยเราจะมี module ทั้งหมดอยู่ 3 แบบ
        local modules
            เป็น module ที่เราสร้างเองใน application ของเรา
        built-in modules
            เป็น module ที่่มากับ nodejs อยู่แล้ว
        thrid party module 
            เป็น module ที่ถูกพัฒนาโดย developer คนอื่น และเรา
            สามารถนำมาใช้ในแอพของเราได้
